Constants

Routines

global function iif(integer test, object true, object false) : object -- inline if statement

Parameters:

  1. test: expression which return a Euphoria true/false
  2. true: the value to return if the expression is true
  3. false: the value to return if the expression is false

Returns:

an object: either true or false
